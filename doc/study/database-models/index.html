Mini curso visual de modelos de base de datos

Recorrido guiado por el modelo de datos del proyecto, dividido por páginas y con cobertura explícita del archivo fuente principal.

🧩 Django ORM 🐘 PostgreSQL 🌐 API 42 🔄 Sync local 📚 Referencias al repo

Objetivo del curso

Estudiar el modelo de datos paso a paso y cubrir todo lo importante del archivo fuente sin dejar huecos. Cada lección mezcla visuales, sintaxis, riesgos, pseudocódigo y preguntas de evaluación.

Ruta recomendada

Empieza por el mapa global, sigue con auth vs dominio, después histórico, incremental, datos propios de la app, sintaxis ORM, migraciones y vistas conectadas.

Tipo de guía

HTML autocontenidos, sin dependencias externas, orientados a repaso rápido en navegador y defensa oral.

Material de apoyo

El PPTX sigue el mismo orden del mini curso y sirve como repaso corto antes de una evaluación.

Cómo abrir y estudiar la guía

Todo es autocontenido. No hay iframes, frameworks ni acceso a la app real.

  • Paso 1: abre doc/study/database-models/index.html en el navegador.
  • Paso 2: sigue el orden 01 → 10 y usa Anterior / Siguiente.
  • Paso 3: completa el quiz de cada página antes de pasar a la siguiente.
  • Paso 4: usa database-models-study.pptx como repaso corto final.

Referencia: doc/database-models-explained.md, sección 1–12 resumen global del documento, línea aproximada 3

Fuente principal obligatoria: doc/database-models-explained.md. Verificación adicional contra modelos, migraciones, rutas y stores reales del repo. Página autocontenida y sin dependencias externas.

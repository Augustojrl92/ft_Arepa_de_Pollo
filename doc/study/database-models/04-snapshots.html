Qué resuelven los snapshots

Sin snapshots solo tendrías el presente. Con snapshots puedes calcular tendencias y comparar evolución sin depender de llamadas en vivo a la API externa.

unique_together, indexes y Meta

Son el cinturón de seguridad del histórico.

unique_together

Impide duplicar dos snapshots para la misma entidad en el mismo día.

class Meta:
    unique_together = ('coalition', 'snapshot_date')

indexes

Aceleran búsquedas por entidad y fecha, justo el patrón más habitual en análisis histórico.

indexes = [
    models.Index(fields=['coalition', 'snapshot_date']),
]

Error común

Pensar que el snapshot reemplaza al modelo actual. No: el snapshot registra la foto temporal; el modelo principal sigue siendo la fuente del presente.

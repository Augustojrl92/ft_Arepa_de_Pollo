Coalition: el estado actual de cada coalicion

Coalition es la entidad que concentra nombre, slug, imagen, color y total_score actual.

Campos clave

coalition_idnameslugcolorimage_urlcover_urltotal_scoreleader_user_idupdated_at

Idea importante

Coalition no guarda historico. Solo la foto actual. Para tendencia 24h/7d/30d hacen falta CoalitionScoreSnapshot.

FUNCIÓN actualizar_coalition(payload):

    buscar por coalition_id
    actualizar nombre, slug, color e imagenes
    actualizar total_score actual
    guardar updated_at

Relacion con usuarios

En este repo la relacion con usuarios no esta normalizada con ForeignKey directa desde CampusUser a Coalition; se resuelve con campos denormalizados como coalition_id, coalition_name y coalition_slug.

Es una decision pragmatica: el sync guarda datos de coalicion embebidos dentro de CampusUser y los servicios/backend reconstruyen la relacion para ranking y vistas.

Como se refleja en las vistas

La pantalla /coalitions consume el endpoint /api/coalitions/ y luego pide detalles con /api/coalitions/details/?coalition=<slug>.

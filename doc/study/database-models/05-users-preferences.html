05. FriendsList y UserPreferences

Esta pagina cubre los datos propios del producto: relaciones sociales, avatar custom, tema, privacidad y preferencias de UX.

Django ORM PostgreSQL API 42 Sync Study mode

Datos propios de la app

FriendsList y UserPreferences no vienen de la API de 42. Son capa de producto local.

🤝

FriendsList

  • owner = OneToOne con User
  • friends = ManyToMany simetrico
  • requests sent / received
  • soporta la capa social del producto
⚙

UserPreferences

  • user = OneToOne con User
  • custom_avatar
  • items_per_page
  • theme_mode, privacy y notifications

Diferencia con datos de 42

Los datos de 42 describen el campus. Los datos de users describen el producto.

Lo que viene de 42

  • login, intra_id y display_name
  • avatar base del campus
  • nivel, score, rank y coalicion
  • metricas de proyectos y correcciones

Lo que vive solo aqui

  • lista de amigos
  • solicitudes pendientes
  • avatar custom subido por el usuario
  • tema, paginacion y privacidad
Frase corta para evaluacion: FriendsList y UserPreferences son producto puro; no forman parte de la API 42.
